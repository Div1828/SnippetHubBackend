SnippetHub – Code & Content Snippet Manager
SnippetHub is a full-stack web application for saving, editing, organizing, and collaborating on code snippets, text notes, and content blocks.

Live Demo

Frontend (Vercel): https://snippet-hub-six.vercel.app/
Backend (Render): https://snippethubbackend.onrender.com

GitHub Repositories

Frontend Repository: https://github.com/Div1828/Snippet-Hub
Backend Repository: this one

Features

Core Features

User Authentication (JWT-based login/register)

Create, Edit, Delete Snippets

Pin/Unpin Snippets

Organize by Category

Search & Filter Support

Public & Private Snippets

Collaborators Support

Real-Time Features

Live Collaborative Editing (Socket.io)

Presence Detection — show who is active on a snippet

Real-Time Syncing of Content and Order

UX & Tech

Frontend: React + TypeScript + Chakra UI

Backend: Node.js + Express + MongoDB (Mongoose)

LocalStorage Support (for unauthenticated features)
